# Tomnium test app

## Available Scripts

In the project directory, you can run:
### "Install"

$ clone https://github.com/Vldkvll/Tomnium-tst.git 


### `Run Server`

$ npm install -g json-server

### `Then`

$ yarn server
or
$ json-server public/db.json --port 8000

### `Run App`
$ npm start
$ yarn start

Open http://localhost:3000 to view it in the browser.
